# Love is puzzle gift server

This folder is the static public gift server.

Publish these files to GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any HTTPS static hosting:

- `gift.enc.json` - encrypted gift data consumed by the Android app.
- `index.html` - a simple status page.

The Android app currently loads:

`https://sudnikalexander-cmyk.github.io/love-is-puzzle-gift/gift.enc.json`

Do not publish local plaintext gift source files.
